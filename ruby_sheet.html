Strings

A string contains one or more characters as text. Strings must always be surrounded by single (') or double (") quotation marks. The characters in a string can be numbered starting from 0, and substrings or characters accessed by these indices. Strings, unlike symbols, are mutable and can be overwritten.

Useful string methods!

  • #capitalize/capitalize! : capitalizes the string/li>
  • #upcase/#upcase!, #downcase/#downcase! : makes every character upper or lower case
  • #length : gets us the length in characters of the string
  • #reverse/#reverse! : reverses the string
  • #to_i : converts a string (of digits) into an integer
  • #to_sym, #intern : converts a string into a symbol

Integers & Floats

These are the basic numerical data types in Ruby. Integers are whole numbers while floats contain a decimal point. (1 is an integer, but 1.0, 1.1, 1.2, etc are floats.) Remember when dividing integers, the nearest whole number rounded down will be returned. Integers and floats can be converted to strings with the #to_s method.

Arrays

These are data structures that contain objects with an ordered integer index, counting from 0. An array is bracketed by square brackets []. Arrays can contain more than one type of object and the objects can be anything.

Useful array methods!

  • #each
  • #each_index
  • #map/map! & synonymous #collect/collect!
  • #cycle
  • #flatten
  • #include?
  • #join
  • #length
  • #pop
  • #push
  • #reverse
  • #select/#select!
  • #slice/#slice!
  • #sort/#sort!

Hashes

A kind of data structure made up of key-value pairs, in which the keys are usually strings or symbols and the values are any kind of object.

Useful hash methods! Many that are shared with the Array class come from the Enumerable module.

  • #each
  • #each_key, #each_value
  • #has_key?, #has_value?
  • #key/#key!
  • #select/#select!
